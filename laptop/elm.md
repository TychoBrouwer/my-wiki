## Install Elm

Get Elm files

```(shell)
wget -O ~/.devtools/elm.gz \
    https://github.com/elm/compiler/releases/download/0.19.1/binary-for-linux-64-bit.gz

gunzip ~/.devtools/elm.gz
```

Install Elm to bin

```(shell)
mv ~/.devtools/elm /usr/local/bin/elm

chmod +x /usr/local/bin/elm
```
